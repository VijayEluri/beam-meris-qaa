Quasi Analytical Processor

Description

"I/O Parameters" tab:

  • Input product file - The existing product that is to be processed by the Quasi-Analytical Processor.  The input product format must be MERIS L2 with the following included:
    • A band close to 410nm
    • A band close to 440nm
    • A band close to 490nm
    • A band close to 510nm
    • A band close to 560nm
    • A band close to 620nm
    • An l2_flags band
  • Output product file - The destination for the output product of the Quasi-Analytical Processor.
  • Output product format - The formatting of the output product.
  • aph check - yes/no option to check the value of the aph during processing and adjust it if necessary. Disabled for the time being.
  • Log filename prefix - Prefix for the filename of the log.
  • Extra log to ouptut dir - Create a file for the extra log in the output directory.
