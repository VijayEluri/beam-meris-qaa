Quasi-Analytical Processor (QAA) v0.8.3

1.  Installation
To install QAA, place the QAA jar file into the modules folder located wherever
Beam-Visat was installed.  QAA will now be available next time Beam-Visat is
started.  
Note: If Beam-Visat is already running, it will have to be restarted before
QAA will be available for use.
